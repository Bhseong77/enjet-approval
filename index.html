📊 결재라인 규칙

휴가신청서: 직속상사 → 부서장 (2단계)
지출결의서: 금액 분기
  • 100만원 미만: 상사 → 재무회계팀장
  • 100~500만원: 상사 → 재무회계팀장 → 본부장
  • 500만원 이상: 상사 → 재무회계팀장 → 본부장 → 대표이사
구매품의서: 상사 → 구매감사팀장 → 본부장 (Phase 2: 구매모듈 연동)
견적승인: 상사 → 본부장 → 대표 (Phase 2: 영업모듈 연동)
